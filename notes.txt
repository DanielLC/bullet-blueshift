Scripting language oddities:

# can't be used in a string, since it will comment out the end quote.
Using / between integers does integer division, rounding towards zero.
There is currently no elif.
You can technically read arrays and fields, but there's no way to create them or write to them.
Commands cannot be printed across lines.
Commands are case insensitive. Nothing else is. I should probably just make them case sensitive.
